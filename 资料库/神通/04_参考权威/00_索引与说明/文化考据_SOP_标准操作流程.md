# 《玄鉴仙族》神通名称文化考据 · SOP标准操作流程

> **版本**：v1.0
> **适用对象**：《玄鉴仙族》五德位业体系中所有神通/仙基名称的文化考据
> **输出目标**：每个神通条目生成包含7个维度的标准化考据文档

---

## 一、考据总纲

### 1.1 核心原则

每个神通名称的文化考据必须同时走**三条路径**，按优先级顺序执行：

**路径A：字面训诂 + 道统象征（必选）**
- 逐字拆解神通名称中每个汉字的古义、本义
- 联系所在道统的意象、象征、五行属性、位别特性
- 判断命名是否呼应道统的核心精神（如少阴"控摄水火"→调杼柚如调理织机经纬）

**路径B：典故溯源（优先尝试）**
- 搜索是否出自道教典籍、诗歌、民俗典故
- 优先检索：《易经》《道德经》《庄子》《诗经》《楚辞》、唐诗宋词、道教经典
- 注意古称/别称的线索（如"天下革"指向革卦）

**路径C：原著关联（补充验证）**
- 查找原著中该神通出现的上下文描写
- 结合原著场景反推命名意图（如"勿查我"→上巫隐匿自保的巫术特性）

### 1.2 三个判断标准

完成考据后，用以下标准检验质量：

1. **道统一致性**：考据结果是否与该道统的位别（正/蕴/收/藏/革/显/过渡/潜变）特性一致？
2. **命名合理性**：考据得出的典故/意象是否能合理解释为什么用这个名称？
3. **原文可验证**：考据结论是否能与原著中的描写相互印证？

---

## 二、七维考据模板

每个神通条目必须填写以下7个维度（无相关内容的维度可标记"—"）：

| 维度 | 说明 | 检索关键词示例 |
|------|------|---------------|
| **字义训诂** | 逐字解析古义、本义、引申义 | "[字] 古义 本义 说文解字" |
| **道教典籍** | 《道藏》《云笈七签》《道德经》《庄子》《易经》《黄庭经》等 | "[词] 道教 典籍" |
| **中医经典** | 《黄帝内经》《本草纲目》五行五脏理论 | "[词] 中医 五行" |
| **诗歌典籍** | 《诗经》《楚辞》《乐府》、唐诗宋词 | "[词] 典故 诗词" |
| **民俗文化** | 民间典故、谐音避讳、信仰习俗 | "[词] 民俗 典故" |
| **道统象征关联** | 与所属道统的意象、位别、五行如何呼应 | 结合前面5个维度分析 |
| **综合考据** | 整体判断：命名意图、文化渊源、在体系中的意义 | 综合所有维度得出 |

## 三、分步操作流程

### Step 1：提取待考条目

从体系梳理文档中提取待考条目，格式如下：

```
神通名称：[名称]
所属道统：[道统名]
所属体系：[五德/五阴/阴阳/十二炁/三雷/三巫/二祝/并古/独立]
位别特性：[正位/蕴位/收位/藏位/革位/显位/过渡位/潜变位]
古称/别称：[如有]
```

### Step 2：路径A — 字面训诂

对每个字进行训诂：

```
1. 查《说文解字》或在线字典，获取每个字的本义
2. 分析字与字之间的组合含义
3. 联系道统的五行属性、位别特性
4. 得出"字面 + 道统"的初步解读
```

**示例**：沆砀满
- 沆：夜间的水气，露水
- 砀：广大，弥漫
- 沆砀：白气弥漫的样子（《汉书》"西颢沆砀"）
- 满：充满
- 道统关联：寒炁→寒冷之炁→沆砀为白气弥漫的寒象→沆砀满=寒气充满天地

### Step 3：路径B — 典故搜索

按优先级进行多轮搜索：

```
第1轮：搜索 "[神通名] 典故 诗词"
第2轮：搜索 "[神通名] 道教 典籍"
第3轮：搜索 "[神通名] 易经 卦"
第4轮：搜索 "[神通名] 庄子 典故"
第5轮：搜索 "[神通名] 诗经 楚辞"
第6轮：搜索 "[古称/别称] 典故"（如有古称优先搜索古称）
第7轮：逐字搜索 "[字1] [字2] 典故"（拆分搜索）
```

**搜索策略**：
- 2字神通优先整体搜索，再逐字搜索
- 3字以上神通先整体搜索，再拆分关键词搜索
- 有古称/别称的优先搜索古称（古称往往更直接指向典故）
- 注意谐音和通假（如"天齐"通"天齐王"泰山神封号）

### Step 4：路径C — 原著关联

```
1. 在原著中查找神通名称出现的章节
2. 收集上下文描写（前后各100字）
3. 结合原著场景分析命名意图
4. 特别注意：首次出现时的描写通常最有解释价值
```

### Step 5：道统象征关联分析

将前面三个路径的结果与道统特性进行关联：

```
1. 该道统的位别特性是什么？（正/蕴/收/藏/革/显/过渡/潜变）
2. 该道统的五行属性是什么？（木/水/金/火/土/炁/雷/巫等）
3. 该道统在原著中的核心意象是什么？
4. 神通名称如何呼应这些特性？
5. 古称/别称是否与这些特性一致？
```

**位别特性速查表**：

| 位别 | 核心特性 | 命名倾向 |
|------|----------|----------|
| 正位 | 中正、主位、根本 | 取根本、方正之象 |
| 蕴位 | 蓄养、潜藏、待发 | 取蓄养、蕴含之象 |
| 收位 | 收敛、聚集、收束 | 取收敛、归聚之象 |
| 藏位 | 隐藏、伏藏、阴藏 | 取隐藏、伏藏之象 |
| 革位 | 变革、革新、破旧立新 | 取变革、更新之象 |
| 显位 | 显赫、明亮、彰显 | 取显明、彰显之象 |
| 过渡位 | 调和、衔接、中介 | 取调和、过渡之象 |
| 潜变位 | 潜藏、变化、隐逸 | 取潜隐、变化之象 |

### Step 6：综合撰写

按照七维模板撰写考据条目，综合三个路径的结果。

### Step 7：质量检验

用三个判断标准检验：
1. 道统一致性？
2. 命名合理性？
3. 原文可验证？

## 四、已验证的考据模式库

以下是经过用户验证的考据模式，可直接参考：

### 模式1：道教日月对举
- 太阳·郁仪文 ↔ 太阴道统·结璘章
- 出处：《太上洞真五星秘授经》"日名郁仪"，《洞渊集》"月名结璘"
- 特征：文/章对举，郁仪/结璘对举

### 模式2：易经卦象命名
- 坎水·入坎窞 → 坎卦初六"入于坎窞"
- 正木·位从专 → 坤卦六二"至静而德方"
- 兑金·位从孚 → 中孚卦"诚信立身"
- 特征：直接取卦名、爻辞入神通名

### 模式3：庄子典故
- 养生主 → 《庄子·养生主》
- 逍遥游 → 《庄子·逍遥游》
- 太冲观 → 《庄子·应帝王》"太冲莫胜"
- 藏壑舟 → 《庄子·大宗师》"藏舟于壑"

### 模式4：诗词名句
- 不胜寒 → 苏轼《水调歌头》"高处不胜寒"
- 惊鹊乡 → 辛弃疾《西江月》"明月别枝惊鹊"
- 病前春 → 刘禹锡"病树前头万木春"
- 凌云木 → 杜荀鹤"时人不识凌云木"
- 特征：取名句中的关键词或缩略

### 模式5：民俗文化
- 槐荫鬼 → 槐树"木中之鬼"民俗信仰
- 佞无晨 → 《尚书》"牝鸡无晨"典故
- 特征：取自民间信仰或历史典故

### 模式6：字义组合 + 道统象征
- 沆砀满 → 沆（水气）+ 砀（弥漫）= 白气弥漫的寒象 → 寒炁
- 调杼柚 → 调理织机经纬 → 少阴"控摄水火"如调理织机
- 特征：字面含义直接呼应道统特性

## 六、常见陷阱与规避

| 陷阱 | 说明 | 规避方法 |
|------|------|----------|
| **过度解读** | 强行关联不相关的典故 | 坚持"三标准"检验，无可靠出处则标记"待考" |
| **忽视古称** | 只搜正名不搜别称 | 有古称/别称的优先搜索古称 |
| **脱离道统** | 考据结果与道统特性不符 | 必须做"道统象征关联"分析 |
| **遗漏维度** | 只搜诗歌不搜道教 | 按七维模板逐一检查 |
| **望文生义** | 只看字面不查典故 | 路径A和路径B必须并行 |

## 七、批量工作流

对于大批量考据（如100+条目），建议按以下批次推进：

```
批次1：高优先级条目（明显有典故痕迹的）
批次2：五德体系核心条目（木/水/金/火/土）
批次3：阴阳体系条目（太阴/太阳/少阳/少阴/厥阴/明阳）
批次4：十二炁 + 三雷 + 三巫
批次5：二祝 + 并古 + 独立道统
批次6：补漏（未覆盖条目）
```

每个批次完成后更新文档并输出进度报告。
