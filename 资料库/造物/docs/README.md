# 写作规范

供生产新 wiki 页面、审稿、写内链时查阅。

| 文件 | 用途 |
|---|---|
| [`内链规范.md`](./内链规范.md) | 跨命名空间内链格式（`[[造物-XX]]` `[[人物-XX]]` 等） |
| [`分类判定规则.md`](./分类判定规则.md) | 品阶 / 道统 / 标签的分类判定标准 |
| [`wiki模板_造物.wiki`](./wiki模板_造物.wiki) | 造物类页面模板（复制粘贴起手） |

## 三同步铁律（最重要）
每页品阶必须三处一致：info 表 / 首句"是…的[[X]]" / Category。
详见 `WORKFLOW.md` 第 2 节。
