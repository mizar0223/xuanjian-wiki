## 一、命名空间约定（强制）

所有跨 wiki 引用 **必须** 使用 `[[<命名空间>-<实体名>|<显示文字>]]` 格式。

| 命名空间 | 用途 | 示例 |
|---------|------|------|
| `造物-` | 法宝/道具/灵资/灵气/丹药/符箓 等具体造物 | `[[造物-太阴月华|太阴月华]]` |
| `人物-` | 角色（修士、凡人、转世前的人格） | `[[人物-李曦明|李曦明]]` |
| `势力-` | 宗门、家族、洞府、国家 | `[[势力-月华元府|月华元府]]` |
| `地点-` | 山川/湖泊/洞天/秘境 | `[[地点-望月湖|望月湖]]` |
| `道统-` | 修真道统（明阳、太阴、府水、剑道...） | `[[道统-太阴|太阴]]` |
| `神通-` | 法术、神通、功法 | `[[神通-太阴吐纳养轮经|太阴吐纳养轮经]]` |
| `事件-` | 重大历史事件 | `[[事件-南北大战|南北大战]]` |

### 1.1 显示文字规则
- 一般情况：`[[人物-李曦明|李曦明]]`（命名空间+原名）
- 文中原带书名号：`[[造物-太阴月华|【太阴月华】]]`（保留书名号）
- 别名/俗称：`[[人物-李曦明|十二世曦明]]`

## 二、加链原则（强制遵守）

### 2.1 First-Mention-Only（首次提及加链）
**同一页面内，同一目标只在第一次出现时加链**。后续出现保留原文不加。

❌ 错误：
```
[[人物-李曦明|李曦明]]见到了月华，[[人物-李曦明|李曦明]]不禁惊叹，[[人物-李曦明|李曦明]]说...
```

✅ 正确：
```
[[人物-李曦明|李曦明]]见到了月华，李曦明不禁惊叹，李曦明说...
```

### 2.2 不加链的位置
- **章节标题** 内不加（如 `== 名字由来 ==` 不动）
- **基本信息表** 的"道统"行已有链接，不重复
- **Category 标签** 不动（`[[Category:XXX]]`）
- **原文引用块** 内的原著文字 **不加链**（保持原貌）
- **考据备注** 中提及实体时可加可不加，谨慎使用

### 2.3 加链优先级（按页面深度判断）

**L1-L3 深度页**（>3KB，已有丰富内链的页面）
- 仅修补：补建红链对应实体后，把红链对应的命名空间纠正
- 不大规模新增

**L4-L6 底版页**（<2KB，内链稀少的页面）
- 主动添加 3-5 个最重要的内链：
  - 持有者人物（首次提及）
  - 道统（如果首行/表格已有则跳过）
  - 关联法宝（首次提及，最多2个）

### 2.4 不该建为wiki的实体
以下作为术语普通文本即可，不加链：
- 修为等级（紫府前期、筑基中期等）→ 不建独立页
- 通用动词（祭炼、感召、突破等）
- 五行属性（金、木、水、火、土）→ 已有道统页承接
