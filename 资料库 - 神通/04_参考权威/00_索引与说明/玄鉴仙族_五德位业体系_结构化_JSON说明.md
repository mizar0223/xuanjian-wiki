### 体系对象结构

每个体系包含：
- `体系名称`: 体系的名称（如"木德"、"水德"、"十二炁"等）
- `位置列表`: 该体系下所有"位置"的数组

## 使用建议

### Python 示例

```python
import json

# 读取 JSON 文件
with open('玄鉴仙族_五德位业体系_结构化.json', 'r', encoding='utf-8') as f:
    data = json.load(f)

# 获取所有体系名称
system_names = [s['体系名称'] for s in data['体系列表']]
print(system_names)

# 查找特定位置
for system in data['体系列表']:
    for pos in system['位置列表']:
        if pos['位置名称'] == '正木':
            print(json.dumps(pos, ensure_ascii=False, indent=2))

# 统计每个体系的神通数量
for system in data['体系列表']:
    total_shen_tong = sum(len(p.get('神通', [])) for p in system['位置列表'])
    print(f"{system['体系名称']}: {total_shen_tong} 个神通")
```

