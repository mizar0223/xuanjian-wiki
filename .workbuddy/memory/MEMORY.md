# Memory - 玄鉴仙族 Wiki 项目

## 项目概况

- **Wiki站点**: leoshixie.devcloud.woa.com（MediaWiki 1.42 + MySQL 8.0 + Nginx，Docker部署）
- **代码仓库**: https://git.woa.com/leoshixie/xuanjian-wiki
- **内容规模**: ~1099页
- **数据流程**: 数据源 → generate_pages.py → .wiki文件 → upload_pages.py → MW API

## 目录结构关键路径

- `资料库/造物/pages/` — 造物分类目录（02-灵物 ~ 13-法术秘法，编号对应品类层级）
- `pages/人物与势力/` — 人物条目（唯一权威源，20-人物已合并入此）
- `pages/仙基道统/` — 仙基道统条目
- `资料库/造物/docs/` — 分类规范文档（原著体系梳理.md、分类判定规则.md、基本信息工作规范.md）
- `小说原文/玄鉴仙族/` — 完整原文（1463章，~16MB）
- `characters/{角色}/` — 年鉴视频渲染相关（audio/、交付/）

## .wiki文件迁移操作规范

迁移条目到新分类目录时，必须修改3处：
1. 开篇定义行（如 `[[灵宝]]` → `[[功法]]`）
2. 品阶字段（同上）
3. Category标签（如 `[[Category:灵宝]]` → `[[Category:功法]]`）

用 `mv` 移动文件后，必须先 Read 新路径的文件，才能用 Edit 修改内容。
